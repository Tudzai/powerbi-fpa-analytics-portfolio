# Power BI FP&A Analytics Portfolio

This repository is a curated Power BI and finance analytics portfolio built to demonstrate FP&A thinking, data modeling, DAX, dashboard design, QA discipline, and business storytelling.

The portfolio is organized as 20 dashboard products. Projects 01-12 contain built Power BI, HTML, semantic-model, or active build artifacts. Projects 13-20 are planned product briefs that extend the portfolio toward regional FP&A, treasury, SaaS, manufacturing, logistics, ESG, governance, and board reporting.

## What To Review

- `README.md` files inside each project for business context and rebuild notes.
- `model/` and `powerbi/` folders for semantic model, DAX, relationships, and Power BI build assets.
- `data/prepared/` folders for star-schema extracts used by the reports.
- `qa/` folders for reconciliation, validation, and desktop checks.
- `output/` folders for final dashboards, previews, screenshots, or QA-accepted PBIX deliverables where available.

## Notes

All demo datasets are synthetic or portfolio sample data. This repository is intended for portfolio review, not production use.

The GitHub version is curated for review: raw source dumps, local runtime tools, temporary extracts, and bulky QA export sessions are intentionally excluded. Final PBIX files, HTML previews, screenshots, prepared data, model definitions, scripts, and documentation are kept where they help reviewers assess the work.
