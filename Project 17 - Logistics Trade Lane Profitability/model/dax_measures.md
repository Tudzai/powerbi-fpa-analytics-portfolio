# DAX Measures

| Measure | Format | Definition |
|---|---:|---|
| Net Revenue | `$#,0` | Total billed revenue after rebates and credits. |
| Freight Cost | `$#,0` | Carrier buy rate and main freight cost. |
| Fuel Cost | `$#,0` | Fuel surcharge cost charged by carriers. |
| Demurrage Cost | `$#,0` | Demurrage and detention leakage. |
| Claims Cost | `$#,0` | Claims and service failure cost. |
| Total Cost | `$#,0` | Total cost-to-serve for shipments. |
| Gross Profit | `$#,0` | Net revenue less total shipment cost. |
| GP Margin % | `0.0%` | Gross profit divided by net revenue. |
| Shipment Count | `#,0` | Total shipment records weighted by shipment count. |
| TEU | `#,0.0` | Twenty-foot equivalent units for ocean lanes. |
| CBM | `#,0.0` | Cubic meter volume handled. |
| Revenue per Shipment | `$#,0` | Net revenue per shipment. |
| Cost per Shipment | `$#,0` | Cost-to-serve per shipment. |
| Target Gross Profit | `$#,0` | Target gross profit using lane/service margin targets. |
| Margin Gap vs Target | `$#,0` | Actual gross profit less target gross profit. |
| Target Margin % | `0.0%` | Weighted target margin for selected scope. |
| Margin Gap % | `0.0%` | Percentage point gap versus target margin. |
| On-Time % | `0.0%` | On-time shipments divided by shipments. |
| Utilization % | `0.0%` | Average equipment/consolidation utilization. |
| Negative Margin Shipments | `#,0` | Shipments with negative gross profit. |
| Reprice Opportunity | `$#,0` | Gross profit required to bring negative shipments to zero. |
| Bridge Amount | `$#,0` | Driver bridge amount for margin gap explanation. |
| Open Actions | `#,0` | Non-closed commercial/procurement/ops actions. |
| Action Risk Value | `$#,0` | Margin value at risk in the action queue. |
| Recovery Value | `$#,0` | Expected recoverable margin from action queue. |
